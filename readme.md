## 🚀 Getting Started

### Prerequisites

Before using these projects, install:

- XAMPP / WAMP / MAMP
- WordPress
- PHP 8.x or later
- MySQL

---

## 📦 Installation

1. Clone the repository

```bash
git clone https://github.com/<your-username>/wordpress_website.git
```

2. Move the required `.zip` file to your WordPress installation.

3. Login to WordPress Admin.

4. Navigate to:

```
Appearance → Themes → Add New → Upload Theme
```

or

```
Plugins → Add New → Upload Plugin
```

depending on the package.

5. Activate the theme.
